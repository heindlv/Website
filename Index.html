  • <section > <⁄section> = Gruppiert Elemente
  • <article> <⁄article> = bereich für Inhalte
  • <aside> <⁄aside> = Bereich für Zusatzinformation
  • <footer> <⁄footer> = Bereich für Fußbereich
  • <h1> <⁄h1> = Hauptübeeschrift
  • <h2> <⁄h2> = Teilüberschriften
  • <p> <⁄p> = Absatz
  • <br ⁄> = erzwungener Zeilenumbruch
  • <b <⁄b> = Schrift wird fett
  • <i> <⁄i> = Schrift wird kursiv
